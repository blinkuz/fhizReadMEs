MAIN MENU 2019.4

Adds a main menu similar to most games when you hit escape. It will give you the options of resume, change character, logout, and exit. If you're using UCE_Options it will also give you a settings option as well.

INSTALLATION PROCESS

Integration Steps

  1. [Required] Drag and Drop all items from the "Prefab [Add to Canvas]" to your canvas.
  2. [Required] Drag and Drop all items from the "Prefab [Add to Shortcuts]" to your shortcut buttons.
  3. [Required] Drag and Drop all items from the "Scripts [Add to Shortcuts]" to your shortcut gameobject.
  4. [Required] Attach all needed components onto "UCE_UI_MainMenuShortcuts" in your canvas, under Shortcuts.
  5. [Required: Core Change] Open the script "NetworkManagerMMO" and find the following code block:
    6. uiPopup.Show("Disconnected.");
    add this right above it... 
    if (!changingCharacters)
  6. [Required: Core Change] Open the script "UILogin" and find the following code block:
    7. if (manager.state == NetworkState.Offline || manager.state == NetworkState.Handshake)
    add this right above it... 
    if (!manager.changingCharacters)
  7. [Optional] Attach any game window panel you want hidden when UCE Main Menu is opened to UCE_UI_MainMenu script under Close Windows.
