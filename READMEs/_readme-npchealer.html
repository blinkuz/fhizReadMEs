NPC HEALER 2019.5

Allows healing at a Npc (Inn/Healer) with variable or fixed prices. Can remove buffs as well.

INSTALLATION PROCESS

Integration Steps

  1. [Required] Drag and drop all prefabs in UI [Drag to Canvas] onto your canvas.
  2. [Required] Drag and drop all prefabs in UI [Add Content to Npc Dialogue] onto your Npc Dialogue.
  3. [Required] Drag and drop all scripts in SCripts [Add to Npc Dialogue] onto your Npc Dialogue.
  4. [Required] Add ButtonHealer to default Npc Dialouge and link it.

none

Additional Information

  1. Gold Cost - the fixed cost to access this healer, is used as a fixed price when "Scale Cost" is unchecked, otherwise its used as multiplicator.
  2. Scale Cost - scales the gold cost according to the amount of health/mana/buffs the player requires to be healed. This can result in a very high cost very quickly.
  3. Scale cost is calculated by multiplying the "Gold Cost" by the amount of health/mana/buffs that needs to be healed/removed.
  4. Unequip Max Cursed Level - does not work with Scaled cost, use fix cost instead.
  5. To prevent confusion, it is recommended to create one healer per service offered and not to mix all of them into one Npcs.
