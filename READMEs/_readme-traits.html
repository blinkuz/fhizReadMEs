PLAYER TRAITS 2019.4

Adds customizeable character traits to players (right now, all player prefabs are the same). Traits can be assigned during character creation. Traits can modifiy almost everything from attributes to stats, skills, resistances etc.

UCE Tools, UCE Attributes, UCE Character Creation

INSTALLATION PROCESS

Integration Steps

  1. [Required] Make sure to install all other AddOns first, as "UCE Traits" requires them.
  2. [Required] Add the "Resources > UCE Traits" folder to your projects main Resources via drag-n-drop.
  3. [Required] Do not keep duplicate objects of the same name and use only one Resources folder in your project.
  4. [Required] Edit the new "UCE Traits" section on your player prefabs and assign a number of trait points to them.
  5. [Optional] Take a look at the Trait templates that ship with the AddOn, modify them and/or create your own via context menu.

Videos

None

Additional Information

  1. Traits are only available during character creation and cannot be altered afterwards anymore (at least not yet).
  2. Traits allow your players to modify almost every aspect on their characters (stats that is, not visual aspects).
  3. Carefully edit the number of trait points each player prefab has available, as well as the trait cost of each trait.
  4. Assign a unique group ID to traits, to limit your players to just one trait of that group.
  5. Check the "Allowed Classes" section on the Trait template, to limit a trait to a certain class.
