EQUIPMENT STANCES 2019.1

Allows you to add individual animations to your characters, that depend on the current equipment. This enables equipment based stances like "Sword Stance", "Twohander Stance", "Bow Stance" and so on. It it not limited to weapon stances, but it is limited to one stance at a time.

INSTALLATION PROCESS

Integration Steps

  1. [Required] Edit each one of your placer prefabs and add all equipment categories that have a individual animation to "UCE Equipment Stances" List. Use the equipment categories like "WeaponSword", "WeaponAxe" etc.
  2. [Required] Edit the animator of each player prefab and add a new animation state for each equipment category as well, for each animation like ("Idle WeaponSword" or "Walk WeaponSword").
  3. [Required] Also add to the animator the same amount of parameters as bool and name them exactly like your equipment category ("WeaponSword").
  4. [Required] In your animator controller, a transition from "AnyState" to your new state and add 2 parameters to its list: The main parameter (like "MOVING" set to true for a "Run" animation) and the equipment stance (like "WeaponSWord"). By combining those 2, you can set a individual run, idle, walk, attack animation for each equipped item as you like.
