Requirements

· Unity LTS
· uMMORPG3d
· Fhiz Tools AddOn

Description

Confirm Delete gives you a popup confirmation dialog before deleting a character. So your players won't accidentially delete the wrong character, or a character in general.

INSTALLATION PROCESS

Integration Steps

  1. [Required] Add everything from this AddOns "Scripts [Drag to CharacterSelection]" folder to your CharacterSelection UI Element found under your scene canvas.
  2. [Required] Drag and Drop each popup component the script requires, Popup Panel, Popup Close Button, Popup Confirm Button, and Popup Text.
  3. [Required: Core Change] Open the script UICharacterSelection then find the following section below and delete it. (You can also comment it out.)
    4. deleteButton.onClick.SetListener(() => { CharacterDeleteMsg message = new CharacterDeleteMsg{index=manager.selection}; manager.client.Send(CharacterDeleteMsg.MsgId, message); });
