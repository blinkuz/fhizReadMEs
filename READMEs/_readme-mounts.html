MOUNTS 2019.3

This AddOn adds a complex Mount system to uMMORPG3d, that will allow players to ride mounts and travel the map at increased speed. Mounts are summoned by using a special item and can be unsummonend using that item as well. Mounts also come with various options and are able to attack enemies while unmounted.

INSTALLATION PROCESS

Integration Steps

  1. [Required] Add all objects from "Prefabs" to your Network Manager spawnables prefabs list.
  2. [Required] Move all items from this AddOns folder "Items" to your main Resources folder > Items.
  3. [Required] Do not keep a duplicate item of the same name.
  4. [Required] Add a new parameter "MOUNTED" to your animator of your player prefab and assign a animation of your choice to it (Should already be there if you are using latest uMMORPG3d version).
  5. [Required] Create both a new tag and a new layer and call both of them Mount.
  6. [Required] Go to "Edit > Project Settings > Physics" and edit the collission layer matrix like shown in the image below.
  7. [Optional] Edit your player prefab and add the "Skeleton Mount" item to it, as starting item in order to test.
  8. [Optional] Start the game and summon the mount by using the item. Click the summoned mount in order to mount it.
  9. [Optional] Now move around the map at increased speed, click the mount item again to unmount your mount. Click the mount item once more to unsummon your mount.
  10. [Optional] See notes below for more use information.

Additional Information

  • Mounts are bound to special Mount items, you can create those items via the context menu.
  • Each mount consists of 2 game objects, you have to create a game object for both: The unmounted mount as well as the mounted mount. You should use the provided example mounts as a base.
  • Mounts have stats, you can easily edit them in the inspector. Check the inspector settings for more details. Only unmounted mounts have combat stats, as mounted mounts do not participate in combat.
  • Unmounted mounts follow their owner around, just like pets, you can edit the settings in the inspector as well.
  • Only unmounted mounts can participate in combat, make sure to edit the Collision layer matrix as described above or they wont react. Also edit their stats in the inspector.
  • Mounts cannot gain experience or level up, but you can create multiple mount items and multiple mount game objects to create stronger variants.
  • Attention: This feature also requires my PVPZones AddOn installed: It is possible to allow players to attack and use skills while being mounted, but you can also dis-allow that on a per-skill basis. In order to do so, you have to check the "uncastable while mounted" checkbox that now appears next each skill.
  • Be sure to check out the various options of Mounted Mounts, in order to make them move faster, optionally hide the player model, unmount the player when hit and more!
