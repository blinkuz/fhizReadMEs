SUMMON ITEM 1.0

Created and maintained by Fhiz - for both personal and commercial projects, not for sharing or re-sale
For uMMORPG3d only - not supported on uMMORPG2d or uSurvival unless noted otherwise

Importing

· Import into a blank uMMORPG3d project of the latest available version.
· Use AddOn version and matching Tools version, most recent versions of both are always recommended.

Requirements

· Unity Engine
· uMMORPG3d
· Fhiz Tools AddOn

Description

This is the same as the Monster Scroll, but it comes in the form of a Skill. Allows the caster to summon on or more Monsters. Combine with the Realm settings in my PVP Zones AddOn to summon allies instead. Add "DestroyAfter" component to the Monsters to have them dis-appear after a specific amount of time again. Summon Undead / Elemental anyone?

INSTALLATION PROCESS

Integration Steps

  1. [Required] Simply create a new "Summon Item".
  2. [Required] Do not keep a duplicate item of the same name and use only one Resources folder in your project.
  3. [Optional] Add a "DestroyAfter" component to the prefabs that can be summoned and set a time. This will destroy summoned objects after a while again.
  4. [Optional] Combine with my "PvP Zone" AddOn and set the Realm of the summoned objects same as the summoner itself. This allows you to create allies.
