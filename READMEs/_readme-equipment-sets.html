EQUIPMENT SETS 2019.4

Allows to add “Individual, Partial and Complete Set Bonus” to equipment items that kicks-in when the player has one, some or all items of a set equipped. Individual, Partial and Complete Set Bonus can be active at the same time. Each Set Bonus can alter all attributes and secondary stats when using in conjunction with "UCE Attributes" AddOn.

none

Additional Information

  1. A equipment set consists of two parts: The actual equipment items and a "Equipment Set" template. You have to create at least two equipment items and exactly one equipment set template for each equipment set.
  2. Edit your items as usual and assign the "Equipment Set" template to each item that belongs to the same set as well.
  3. There are three different kinds of "set bonus" in total, each equipment set can feature one, some or all of them.
  4. Individual Set Bonus - You edit this bonus on each item itself, so it can be unique on a per-item basis. It's activated on each item and only when the full set is worn.
  5. Partial Set Bonus - This is defined on the "Equipment Set Template" and is activated when a certain amount of items is equipped. You can define the number of set items that have to be equipped (like 4 out of 6 etc.).
  6. Complete Set Bonus - This final bonus is also defined on the "Equipment Set Template" and is activated only when all set-items are equipped (e.g. the full set).
