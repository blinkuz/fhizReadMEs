ATTACHE 2019.3

Allows you to attach game objects to Entities, those game objects can spawn and unspawn together with their "parent". They can also spawn and unspawn according to certain rules (like parent under 50% health or parent above 50% health etc.). You can use this to summon minions to aid your parent monster and/or player. Also useable to add blood drops or smoke to monsters with low health etc.

INSTALLATION PROCESS

Integration Steps

  1. [Required] Simply edit any Entity in your scene or in prefabs, see the new Attache section in it's Inspector.
  2. [Note] This AddOn is a bit complicated to understand and use at first, but it is also very powerful!

Videos

Additional Information

  1. "Child Game Objects" are the game objects that can be spawned, when the parent meets certain criteria.
  2. Make sure that the "Child Game Objects" have been added to your Network Managers list of spawnable prefabs.
  3. Child Objects will be spawned at the "Mount Point" of its parent, use the default Mount Point or create a new one on you parent object.
  4. Auto Spawn - the child object will be spawned immediately after the conditions are met (does not apply when child is set to "destroy").
  5. Child Conditions - you can edit all the conditions the parent objects has to meet here, in order for the child to be spawned.

You can add more than one child game object per "Attached Child" entry. It will then spawn a child up to the level of the parent object. Example: Parent is level 5 and you enter 10 child objects, it will then spawn child object number 5.
