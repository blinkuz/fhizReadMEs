CHARACTER CREATION 2019.6

Adds a character creation interface to the game that replaces the default interface. It allows you to look at the available classes while creating a new character.

INSTALLATION PROCESS

Integration Steps

  1. [Required] Move the "CharacterCreationSpawnPoint" to any area of your scene. This will be where the preview of the player model is created. It should be a special area way outside of the real playable region of your game.
  2. [Required] Move the "Creation Transform" to that area as well and make it look at your "CharacterCreationSpawnPoint".
  3. [Required] Add the "UCE_UI_CharacterCreation" UI element to your canvas. Make sure to put it directly above the default "Character Selection" in your canvas.
  4. [Required] Check the "Class List" and expand the list to match the number of classes available in your game.
  5. [Required] Add buttons and text to each entry in the "Class List". If you have more than 2, add a new button and link that button and its text as well.
  6. [Required] Link the "SpawnPoint" to the "CharacterSpawnPoint"
  7. [Required] Link the "Network Manager" to the Network Manager in your scene.
  8. [Required] Link the "Creation Camera Location" to the "Creation Transform" in your scene.
  9. [Required] Finally make sure to deactivate the old Character Creation UI element. Do not delete them (we might need them later for updates).
  10. [Required] Now link the new "UCE_UI_CharacterCreation" in the default CharacterSelection in the inspector. To activate the new Character Creation panel instead of the default one.

Additional Information

If you check "Look at Camera" it will force the model to always look towards the "Creation Transform". Uncheck if not required.
