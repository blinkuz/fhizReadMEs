TAUNT ITEM 2019.0

Created and maintained by Fhiz - for both personal and commercial projects, not for sharing or re-sale
For uMMORPG3d only - not supported on uMMORPG2d or uSurvival unless noted otherwise

Importing

· Import into a blank uMMORPG3d project of the latest available version.
· Use AddOn version and matching Tools version, most recent versions of both are always recommended.

Requirements

· Unity Engine
· uMMORPG3d
· Fhiz Tools AddOn

Description

A new item type that will taunt the current target, if it is a Monster. Simple and effectively create a Bell to attract monsters etc.

INSTALLATION PROCESS

Integration Steps

  1. [Required] Move the from this AddOns Resources > Items folder into your projects main Resources > Items folder.
  2. [Required] Do not keep a duplicate item of the same name and use only one Resources folder in your project.
  3. [Required] Add the example item to anyone of your player prefabs.
  4. [Optional] Edit or create a new taunt item.
