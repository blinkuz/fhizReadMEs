INSTALLATION PROCESS

Integration Steps

  1. [Required] Assign to the "Projectiles" the "Impact Object" you want (use the provided one as example), also set the "Spawn Chance" and other settings.
  2. [Required] Check the "Prefabs" folder of this AddOn and add all items therein to your NetworkManager registered spawnables prefab list.
  3. [Required] Create a new projectile skill and add one of the projectiles in this AddOns prefabs folder to it.
  4. [Required] Edit the rest of the skill settings in the inspector, add a animation state and parameter to your player prefab.
  5. [Required] Make the skill available to your players in the game (for example to add it to the skill list and check "learned by default" on the skill) in order to test it.
  6. [Note] Check the tooltips on the skill itself for more information
  7. [Note] If the textures on the sample projectiles show up in pink, please either install unity standard assets or replace them by using a effect pack from the asset store
  8. [Note] Ballistic and Collider Projectiles require you to set Physics layer: "Monster VS. Ignore Raycast" to true (check the box in Physics Layer settings)

Additional Information

  1. Note: When using a muzzle or impact skill effect, you also have to add a "OneTimeTargetSkillEffect" component to it (like in default uMMORPG).
  2. All important settings are on the Skill itself, you only edit those. Only edit the projectile prefabs if you want to change how they look.
  3. The projectile skill is fully functional with all "UCE projectile" prefabs.
  4. To create a new projectile type, simply duplicate one of the "UCE projectile" prefabs.
  5. ArrowProjectile - functions like a ranged skill in uMMORPG with extra options but no physics. Destroyed on Impact. Low Network Traffic & Calculations.
  6. ColliderProjectile - a physics based projectile with a collider. Allows for penetration projectiles. Destroyed after travelled distance/lifetime. Medium-High Network Traffic & Calculations.
  7. BallisticProjectile - A physics based projectile with a collider. Allows for penetration projectiles & arced fly-path. Destroyed after travelled distance/lifetime. High Network Traffic & Calculations.
