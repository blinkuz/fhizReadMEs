SKILL - CONVERT 2019.1

A new Skill type that will convert the Realm and alliedRealm of the current target to your own, if it is a Monster. Comes with the usual skill options like animation, mana cost and range - but also has a success chance per level. Simple and effectively create a Charm skill etc.

INSTALLATION PROCESS

Integration Steps

  1. [Required] Simply create a new Skill via the context menu and edit it's settings in the inspector.
  2. [Required] Make the skill available to your players in order to test.
  3. [Required] Switch to the "Animator" of your player prefab(s) and add a new Animator state and name it just like your Skill.
  4. [Required] Still on the "Animator" add a new Animation Parameter as Bool. Name as skill.
  5. [Required] Still on the "Animator" create a transition and new Animator state to the rest of the Animator.
  6. [Required] Still on the "Animator" set the conditions on the transition (see how the other skills do it) and add your new Bool Parameter (true) to it.
