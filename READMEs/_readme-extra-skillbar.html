UCE EXTRA SKILLBAR 2019.1

Allows you to create equipment specific extra-skillbars.

Adds an extra skillbar above the default one, assign keys to it via the player prefabs. Each weapon can have an ID number now. When changing equipment the extra skillbar of the corresponding ID is shown.

Now players can assign skill sets to these extra skillbars and when switching weapon, the correct skillbar is shown. Example: Equipping an "Axe" shows skillbar ID 1. Then the player can add "Axe skills" to that skillbar. Whenever switching to an Axe, the skillbar 1 is shown together with all axe skills assigned to it.

INSTALLATION PROCESS

Integration Steps

  1. [Required] Add the new skillbar prefab to your canvas, place it above the default one.
  2. [Required] On each equipable item, there is now a "skillbarIndex", set to 0 to disable the skillbar for this item.
  3. [Optional] Play a bit with the settings: When equipping an item with skillbarIndex greater than 0, it will show that skillbar and the player can assign skills to it. When equipping a item with index 0, it will hide the skillbar again.
  4. [Note] All extra skillbars are saved client side, so the player could setup a skillbar for each index number (1=axes, 2=swords, 3=daggers and so on).

Additional Information

Of course, this AddOn only makes sense with Weapon items, but it works with every equipment slot. You should just be careful that the player is not able to equip several items with a skillbar index at the same time, as this will mess up the system.
